Warrior's Spirit

  • The first mod I ever created. It simply added a new skill to the game Dungeons of Dredmor called Warrior's Spirit. It was a skill that helped out warrior type builds in the game. It was mainly a passive skill list to be used in conjunction with other skills. It received over 1,000 downloads during the time it's been released.
  • Release Date: June 6, 2012
  • Platform: PC
  • Base Game: Dungeons of Dredmor
  • Mod Link: The mod is free to download here (requires Steam and a copy of Dungeons of Dredmor to play).

The Mighty Duur Race

  • The Mighty Duur Race is a mod that adds a new species to the space strategy game Stellaris. It started out as an inside joke among friends, 'what if there were space deers that traveled the galaxy?'. Anyways this mod adds a pre-built empire that can spawn in peoples game as well as plenty of deer portraits that can be used for players to create their own deer empire in space. There are several more features I would like to add if I have the time such as custom events but it is fully playable in it's current state.
  • Release Date: January 18, 2017
  • Platform: PC
  • Base Game: Stellaris
  • Mod Link: The mod is free to download here (requires Steam and a copy of Stellaris to play).

Through The Forest

  • Through The Forest is a 18 hole mini golf course for the game Golf It! It is an easier course that tries to be fair. This was mainly to try my hand at designing a full course that others could play and not get frustrated.
  • Release Date: July 12, 2017
  • Platform: PC
  • Base Game: Golf It!
  • Mod Link: The mod is free to download here (requires Steam and a copy of Golf It! to play).
